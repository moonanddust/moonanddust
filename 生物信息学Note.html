  • 旁系同源：一个物种内由同一祖先通过基因复制而来的基因，可能会进化出新的但与原功能相关的功能

  • DNA组装

    • 定义：

    • 遇到的问题

      1. 碱基标识错误（测序错误）

      2. 未知方向

        • 增大测序的长度

        • 增大测序的深度，增加测序量

  • 片段组装模型

    1. 最短公共超串模型

      • 不能解决碱基标识错误，且必须已知方向

    2. 重构模型

      • f作为g的近似字串的代价
      • 不同 ϵ 有不同的组装结果

    3. 多重连续区模型

      1. OM图 (overlap multiple graph)

        1. 节点：子序列
        2. 边：两条序列存在覆盖
        3. 方向：从后缀指向前缀
        4. 边的权值：覆盖强度（有几个碱基）

      2. 汉密尔顿路径

        1. 有且仅有一次通过每一个节点的路径

        2. |S(P)|=||A||−w(P)

          1. 最短超串长度，所有片段长度，权值和

        3. 所以求最短公共超串就是求权值最大的汉密尔顿路径

      3. 定理与推论

      4. 片段组装的一般过程

        1. 删除子串
        2. 组建OM图
        3. 寻找最高权值的汉密尔顿途径
        4. 组装

      5. 如何寻找汉密尔顿途径?

        1. 贪心算法 greedy algorithm

          1. 简化OM图，顶点间仅考虑权值大的边，得到OG图 (overlap graph)
          2. 删除最大权值的边，删除后缀节点发出的其他边和前缀节点进入的其他边
          3. 重复这个过程，直到选出所有的有向边

        2. 拓扑排序算法 topology order algorithm

          1. 建立OG图
          2. 选择入度为0的节点，然后删除这个节点发出的所有边
          3. 重复这个过程，直到选出所有的有向边

预测方法

  • 经验参数法

    • 蛋白质二级结构有强烈规则

    • 蛋白质中3个二级结构含量占85%

    • 每种二级结构的分布不均一

    • 倾向因子

      • Pi=Ai/Ti (i=α,β,c,t)
      • Ti 是所有被统计的残基中处于构象i的比例
      • Ai 是第A中残基处于构象i的比例
      • 若Pi>1.0，说明倾向于形成二级结构构象i

  • 二级结构的经验规则

    1. α helix rule

      • 找到螺旋核：相邻的6个残基中如果有至少4个残基倾向于形成α螺旋，则认为是螺旋核
      • 延伸：从螺旋核向两端延伸直至四肽片段的α螺旋倾向性因子的平均值Pα<1.0为止
      • 两端各去掉3个残基，剩余部分若长于6个残基，而且Pα>1.03，则预测为螺旋

    2. β sheet rule

      • 思路同上
      • 相邻6个残基中若有4个倾向于形成β折叠，则认为是折叠核。 折叠核向两端延伸直至4个残基的平均折叠倾向性因子Pβ<1.0。 若延伸后的片段的 Pβ>1.05，则预测为β折叠

    3. turn rule

      • 判断4个连续残基的概率值乘积是否大于 7.5×10−5

    4. overlap rule

      • 若α和β都满足，比较Pα和Pβ，哪个大就是哪个

  • GOR（Garnier-Osguthorpe-Robson）

    • GOR将蛋白质序列当作一连串的信息值来处理

    • 基于信息论和贝叶斯统计

    • 考虑被预测位置氨基酸种类的影响，也考虑相邻残基种类对该位置构象的影响

    • 定义：I(S;R)=log[P(S | R)/P(S)]

      • 含义：

        • R代表中心氨基酸及其所处环境
        • S代表二级结构类型
        • I(S;R)代表中心氨基酸处于S的信息值

      • 关系：

        • 若S和R无关，则I(S;R)=0
        • 若R的发生有利于S的发生，则I(S;R)>0
        • 若R的发生不利于S的发生，则II(S;R)<0

  • Lim method（立体化学方法）

    • 根据疏水氨基酸来判断

    • α 螺旋的形成规律：

      • 在一段序列中发现第i、i+3、i+4位是疏水残基时，这一片段就被预测为α螺旋
      • 当发现第i、i+1、i+4位为疏水残基时，这一片段也被预测为α螺旋

    • β 螺旋的形成规律：

      • 埋藏的β 折叠通常由连续的疏水残基组成
      • 一侧暴露的β 折叠则通常具有亲水-疏水的两残基重复模式

    • 点模式

      • 将20种氨基酸残基分为亲水、疏水以及两性残基三类
      • 用八残基片段表征亲疏水间隔模式
      • 以一个二进制位代表一个残基，疏水为1，亲水为0，共八位
      • 这样，八残基片段的亲疏水模式可用0-255的数值来表示

  • 同源分析法

    • 将待预测的片段与数据库中已知二级结构的片段进行相似性比较，计算相似性得分，基于此构建二级结构
    • 通常，同源性大于30%，预测结果更好

  • 综合方法

    • 多个程序同时预测，综合评判得到一致结果

    • 序列比对与二级结构预测

    • 双重预测

      • 首先预测蛋白质的结构类型，然后再预测二级结构

Protein 3D structure prediction

  1. 同源建模方法

    • 适用条件：两个蛋白的相似性大于30%

    • 思想：为未知结构寻找已知的结构的同源蛋白质模版

    • 理论：如果两个蛋白的identity超过30%，那么它们就有相似的结构（超过60%就非常相似了）

    • 步骤：

      1. 搜索结构模型的模版
      2. 序列比对
      3. 构建骨架
      4. 构建目标蛋白质的侧链
      5. 构建目标蛋白质的loop区
      6. 优化模型

  2. 折叠识别法/穿针引线法

    • 适用条件：远程同源，即两个蛋白质有相似结构，但是序列同源性低于25%

    • 未知结构蛋白——T；已知结构的远程同源蛋白——U

    • 需要解决的问题：

      1. 如何找到远程同源蛋白
      2. 正确比对U和T
      3. 构建远程同源模型

    • 如何解决1和2？

      • 建立一个U->T的线索，并且通过一些基于环境或基于知识的势函数，评价序列与结构的适应性

    • 线索化 threading

      • 主要思想：利用氨基酸的结构倾向（如形成二级结构的倾向、疏水性、极性等），评价一个序列所对应的结构是否能够适配到一个给定的结构环境中

      • 定义：建立序列到结构的线索的过程

      • 目标：为目标蛋白质U寻找合适的蛋白质模板，这些模板蛋白质与U没有显著的序列相似性，但却是远程同源的

      • 步骤：

        1. 构建模版库

        2. 设计打分函数

          • E=Ep+Es+Em+Eg+Ess

        3. 序列结构比对

        4. 折叠识别和模型构建
